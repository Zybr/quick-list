### Download project
```shell script
mkdir quick-list
cd quick-list
git init
git remote add origin git@github.com:Zybr/quick-list.git
git branch --set-upstream-to=origin/master
git pull
```

### Install packages
```shell script
yarn install # Local packages
```
### Start
```shell script
yarn start # Start server
```
### Tests
```shell script
yarn test # Start server
```
